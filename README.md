# About-Hirasawa
本项目最早只是想解读平泽进的Solo曲目  
而因为现存翻译质量参差不齐, 解读非常困难, 迫不得已进行了重翻译  

本项目翻译优先: 简洁/还原原意/与原文语序尽量一致(以防割裂感)  
翻译的美感并非重点, 只在遵循上三条的情况下尽可能美化  
目标是能让听众将翻译与原文/歌声联系起来, 听多了能达到不需要翻译的程度  

核P-MODEL专辑里带有实体专辑/会员杂志上的内容, 并非由我撰写  
Solo后期专辑翻译都会带有导读用于描述曲目在专辑里扮演的角色, 内容都是我的主观猜测  
且导读的内容只是千万种解释的其中一种, 请不要被限制了想象力  

语序对齐有时会导致出一些比较奇怪的翻译, 以及有些地方实在无法对齐请注意  
双引号有多重含义, 请根据上下文分辨  
且因很多地方的'と'翻译不出, 则直接用双引号表示是某人说的话并将'と'省略了  

所有翻译/导读都可能会改动, 详情可以看Commit  
如果发现翻译有误/有更好的翻译方法可以提交Issue或Pull Request  
还没翻译的专辑你也可以直接翻译后提交PR  
有疑问/想要讨论可以加下方的Discord频道
