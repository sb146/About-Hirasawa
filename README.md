## 翻译准则
本项目翻译优先: 简洁/还原原意/与原文语序尽量一致(以防割裂感)  
翻译的美感并非重点, 只在遵循上三条的情况下尽可能美化  

目标是能让日文不是很好的听众能将翻译和歌声/原文联系起来, 最好能达到不需要翻译的程度  
我希望此项目的翻译只是作为你理解原文的桥梁/过渡, 如果比较在乎平泽进的这些内容的话, 请一定要试着看原文  

## 翻译细节
语序对齐有时会导致出一些比较奇怪的翻译, 以及有些地方实在无法对齐请注意

1. 双引号有多重含义, 请根据上下文分辨
2. 很多地方的'と'翻译不出来, 则直接用双引号表示是某人说的话并将'と'省略了
3. "nをv" 一般意为 "将n v", 译文会把'将'前置, 有时也会翻译为 "v了n" / "v着n" 以及其他
4. "nでv" 一般意为 "以n v" 或 "在n v" 以及其他, 译文会前置
5. "nへ" 一般意为 "向着n", 译文会前置
6. 'よ'作为语气词有时会翻不出来并省略掉, 请注意
7. "...を見た" / "...を聞た"以及类似的会翻译为"看见了..." / "听见了..."
